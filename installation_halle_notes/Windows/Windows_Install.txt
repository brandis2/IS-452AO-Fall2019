Anaconda

- Make sure your correct system is selected

- Do not need to sign up for Anaconda Cheat Sheets
- Installer will pop up, leading you through the installation process
    - Under advanced options, you will select first box despite it getting angry (See elizabeth instructions for
    screenshot)
- Takes a while to install
- It will ask about Microsoft Visual Code. You do not need to download this. (If you do have this already downloaded
    do not use)
- Open command prompt (screenshot); type "python" (without quotes) and enter. Make sure it says "Python 3.6" underneath
